## 👋 About / Philosophy

**初めて触れる人でも迷わず使え、  
自然とまた使いたくなる体験をつくる。**

そんな体験を目指して、Webアプリケーションの開発に取り組んでいます。

技術そのものだけでなく、  
「誰が・どんな状況で・どのように使うのか」を起点に考えることを大切にしています。

UI / UX を軸にフロントエンドを中心としながら、  
バックエンドやデータベースにも触れ、  
設計から実装まで一貫して関わる開発を行ってきました。
